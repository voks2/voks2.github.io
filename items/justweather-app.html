JustWeather is a lightweight web app that provides current weather updates and forecasts for the user’s current location and any other location of their choice.

This app was primarily developed to demonstrate product documentation practices and to explore the integration of AI in coding workflows. While AI tools supported the development process, all functionalities were manually tested and refined to ensure quality and reliability.

Features
  • Real-Time Weather Data: Integrates with the OpenWeather API to provide accurate and up-to-date weather information for any selected location.
  • Automatic Location Detection: Enables weather updates for the user’s current location, based on geolocation permissions.
  • Custom Location Search: Utilizes the HERE API to allow manual entry of locations for tailored weather forecasts.
